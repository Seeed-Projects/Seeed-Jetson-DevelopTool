# GUI Localization Terminology

This project keeps product, SDK, protocol, package, and command names unchanged
across GUI locales:

- Jetson, Orin, JetPack, L4T, BSP, CUDA, TensorRT, cuDNN, DeepStream, VPI,
  Riva, Isaac, NGC, SDK Manager, Docker, PyTorch, Jupyter Lab, VS Code, SSH,
  VNC/noVNC, RTSP, GMSL, NVMe, HDMI, USB, SHA256, jtop

## Jetson Hardware Terms

NVIDIA localized Jetson pages use these terms for developer kits:

- French: `kit de développement`
- Spanish: `kit de desarrollador`
- German: `Entwicklerkit`

## UI Defaults

| English | French | Spanish | German |
| --- | --- | --- | --- |
| Flash / Flashing | Flashage / Flasher | Flasheo / Flashear | Flashen |
| Recovery Mode | Mode de récupération | Modo de recuperación | Wiederherstellungsmodus |
| Device | Appareil | Dispositivo | Gerät |
| Remote Development | Développement à distance | Desarrollo remoto | Remote-Entwicklung |
| Diagnostics | Diagnostic | Diagnóstico | Diagnose |
| Peripherals | Périphériques | Periféricos | Peripheriegeräte |
| Deploy | Déployer | Desplegar | Bereitstellen |
| Firmware | Firmware | Firmware | Firmware |
| Boot Disk | Disque de démarrage | Disco de arranque | Boot-Datenträger |
| Headless | Sans écran (headless) | Headless | Headless |
| Subnet | Sous-réseau | Subred | Subnetz |
| Gateway | Passerelle | Pasarela | Gateway |
| Serial Port | Port série | Puerto serie | Serieller Port |
| Wheel (Python) | wheel | wheel | Wheel |

Product names (Orin Nano, Orin NX, Jetson Nano, reComputer, etc.) are not
translated. Service names (x11vnc, noVNC, Docker, Ollama, Node.js) and CLI
identifiers (claude, codex, openclaw, aria2c, lsusb, jtop) are kept verbatim.
